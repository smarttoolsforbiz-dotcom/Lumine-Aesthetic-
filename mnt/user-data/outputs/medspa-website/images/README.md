# Images Directory

This directory should contain all images for your medical spa website.

## Quick Start

Replace placeholder images with your own professional photos. See `IMAGE-GUIDE.md` for detailed specifications.

## Directory Structure Created

- `/services` - Treatment/service images
- `/results` - Before/after photos
- `/about` - Team and facility photos
- `/testimonials` - Client photos
- `/instagram` - Social media content
- `/badges` - Certification badges

## Image Requirements

- **Services**: 800x600px JPG
- **Results**: 600x800px JPG (before/after pairs)
- **Team**: 800x800px JPG
- **Testimonials**: 300x300px JPG
- **Instagram**: 1080x1080px JPG
- **Badges**: SVG or PNG with transparency

## Optimization

Before adding images:
1. Resize to correct dimensions
2. Compress using TinyPNG or similar
3. Target < 200KB per image

For detailed specifications, see `IMAGE-GUIDE.md` in this directory.
